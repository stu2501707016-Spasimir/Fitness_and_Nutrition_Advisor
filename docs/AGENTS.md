# Агенти

## UserIntakeAgent
Проверява валидността на входните данни.

## BodyProfileAgent
Изчислява BMI и избира BMI категория.

## OntologyMappingAgent
Свързва UI отговорите с individuals от онтологията.

## OntologyReasoningAgent
Добавя временен runtime user и извлича inferred class assertions.

## NutritionRecommendationAgent
Оценява NutritionPlan individuals по цел, BMI, опит, режими и ограничения.

## TrainingRecommendationAgent
Оценява TrainingPlan individuals по цел, опит, среда, продължителност, честота и оборудване.

## ExplanationAgent
Генерира разбираемо обяснение защо системата е избрала плановете.
